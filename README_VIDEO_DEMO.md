## üé® VISUAL GUIDELINES

### Text Overlays:
- **Font:** Sans-serif, clear and readable
- **Size:** Large enough to read on mobile
- **Color:** High contrast with background
- **Position:** Don't obstruct important UI
- **Duration:** Long enough to read (3-5 seconds)

### Highlights:
- Use subtle glow or border
- Don't be too distracting
- Use consistent style
- Highlight important elements only

### Transitions:
- Use smooth transitions
- Don't overuse effects
- Keep it professional
- Match transition style

### Common Issues:

#### Video Quality Poor:
- Increase recording resolution
- Increase bitrate

- Use better recording software

- Close unnecessary applications
- Clear browser cache
- Use faster computer
- Reduce recording quality temporarily

- Practice narration
- Adjust script timing
- Edit out pauses
- Use editing software to adjust speed
